## Installation

clone rep:
```bash
git clone https://github.com/AxmetES/ToDoList.git
```

create virtual environment:
```bash
python3 -m venv venv
```

activate environment:
```bash
source venv/bin/activate
```

install requirements:
```bash
pip install -r requirements.txt
```

## Usage

```bash
python3 main.py
```
